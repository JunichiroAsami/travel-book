## ドキュメントの配置ルール

```
ai_travel_book_project/
├── LATEST.md                     # 最新版ファイル一覧
├── README.md                     # プロジェクト概要
├── planning/                     # 企画・設計
│   ├── book_proposal.md          # 書籍企画書

├── materials/                    # 素材

│   ├── conversation_index_v2.md  # 会話ログインデックス
│   └── chapters/                 # 章別素材集
├── drafts/                       # 初稿
│   └── chapter3_draft.md         # 第3章初稿
├── finals/                       # 最終稿
│   └── chapter3_final.md         # 第3章最終稿
├── project_management/           # プロジェクト管理
│   ├── progress_report_v3.md     # 進捗レポート最新版
│   ├── issue_tracker.md          # 課題管理表
│   └── reports/                  # レポート
└── backup/                       # バックアップ・分析
    └── backup_manual.sh          # バックアップスクリプト
```

### ファイル命名規則

- **バージョン管理**: ファイル名に `_v2`, `_v3` などのバージョン番号を付与
- **章番号**: `chapter1_`, `chapter2_` などの章番号を付与
- **最新版**: バージョン番号が最も大きいものが最新版
