# 3つのAIへの相談プロンプト

**作成日時**：2025年11月27日  
**目的**：第3部と付録Bの重複感を解消するための具体的な実装方法を聞く

---

## プロンプト（共通）

```
# 背景

私は「AIと共に旅する、新しい冒険の形」という書籍を執筆しています。この書籍は、ベトナム・タイ周遊記を通じて、AIを旅行に活用する方法を紹介するものです。

書籍の構成は以下の通りです：

- **第3部（第6章～第9章）**：実践編 - ベトナム・タイ周遊記
  - 第6章：旅行前の準備に使える「6つの型」
  - 第7章：旅行中の情報収集に使える「7つの型」
  - 第8章：トラブル対応に使える「5つの型」
  - 第9章：旅行後の振り返りに使える「4つの型」

- **付録B**：プロンプトテンプレート集
  - 18個の実践的なプロンプトテンプレート（旅行計画編、グルメ編、トラブル対応編、振り返り編）

# 問題

あなたは以前、この書籍をレビューした際に、以下のように指摘しました：

> 第6章〜第9章（実践プロンプト集）と、付録B（プロンプトテンプレート集）は内容が密接に関連しており、読者によっては「同じことが繰り返されている」と感じる可能性があります。読者にとって「どちらを使えばいいのか」が不明確になる可能性がある。

# 現在の対応状況

この指摘を受けて、以下の対応を実施しました：

1. **第2章に「2.3. 第3部と付録Bの使い分け」を追加**
   - 第3部：ストーリーの中で学ぶ（初めてAIを使う方、ストーリーを楽しみながら学びたい方向け）
   - 付録B：辞書的なリファレンス（すぐに使いたい方、特定のプロンプトを探している方向け）
   - 相互参照の活用方法を説明

2. **付録Bの冒頭に「本付録の使い方」を追加**
   - 第3部との違いを表形式で明示
   - 相互参照の活用方法を説明

# 質問

上記の対応で、読者の混乱は解消されると思いますか？

もし不十分な場合、以下のような追加対応が必要でしょうか：

**オプション1：相互参照の追加（詳細版）**
- 第3部の各プロンプトに「詳細は付録B-XX参照」を追加（約50～100箇所）
- 付録Bの各テンプレートに「使用例は第X章参照」を追加（18箇所）

**オプション2：相互参照の追加（簡略版）**
- 第6章～第9章の冒頭に「本章で紹介するプロンプトの詳細なテンプレートは、付録Bを参照してください」を追加（4箇所）
- 付録Bの各テンプレートに「使用例は第X章参照」を追加（18箇所）

**オプション3：現状のまま（追加対応なし）**
- 第2章と付録Bの冒頭の説明だけで十分

あなたの推奨は、オプション1、2、3のどれですか？理由も含めて教えてください。

また、他に効果的な対応方法があれば、具体的に提案してください。
```
