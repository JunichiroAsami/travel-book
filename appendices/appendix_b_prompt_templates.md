# 付録B: プロンプトテンプレート集

**本付録について**

本付録には、旅行中にAIを効果的に活用するための実践的なプロンプトテンプレートを収録しています。本文で紹介したプロンプトエンジニアリングの原則に基づき、すぐに使える形式で整理しました。

## 本付録の使い方

### テンプレートの構成

各テンプレートは以下の要素で構成されています：

- **テンプレート名**: プロンプトの用途
- **使用場面**: どのような状況で使うか
- **プロンプト例**: そのまま使える具体例
- **カスタマイズのポイント**: 自分の状況に合わせて変更すべき箇所
- **期待される出力**: AIからどのような応答が得られるか

### 記号の意味

- **[  ]**: あなたが入力すべき情報
- **{  }**: 選択肢から選ぶ情報
- **※**: 重要な注意事項

## 1. 旅行計画編

### 1-1. 基本的な旅程作成

**使用場面**: 旅行の全体的な計画を立てる初期段階

**プロンプト例**:

```
以下の条件で旅行プランを3つ提案してください：

【基本情報】
- 出発日: [2025年8月12日]
- 帰国日: [2025年8月19日]
- 出発地: [東京（成田空港）]
- 目的地: [ホーチミン、バンコク]
- 旅行人数: [1名]
- 予算: [総額{30万円/50万円/制限なし}]

【希望事項】
- 旅行の目的: [観光、リラックス、仕事、文化体験など]
- 興味のあるアクティビティ: [グルメ、歴史、自然、ショッピングなど]
- 宿泊の希望: [ホテルのグレード、立地の希望など]
- 移動手段の希望: [飛行機、列車、バスなど]

【制約条件】
- 確定している予定: [8月15日夜に友人と食事]
- 避けたいこと: [長時間移動、早朝出発など]
- その他の要望: [自由記述]

各プランについて、以下を含めて提案してください：
1. 日程の概要（表形式）
2. 各日の詳細な行動予定
3. 推奨ホテル（3段階の価格帯で）
4. 移動手段と所要時間
5. 概算費用
6. プランの特徴（メリット・デメリット）
```

**カスタマイズのポイント**:
- [  ]内を自分の情報に置き換える
- 希望事項は具体的に書くほど良い提案が得られる
- 制約条件は漏れなく伝える

**期待される出力**:
- 3つの異なるテーマの旅程案
- 各プランの詳細な日程表
- ホテルや移動手段の具体的な提案
- 費用の概算

### 1-2. 特定の都市の観光プラン

**使用場面**: 特定の都市での1日の過ごし方を計画する

**プロンプト例**:

```
[ハノイ]での1日観光プランを作成してください。

【条件】
- 滞在日: [8月13日（火）]
- 開始時刻: [9:00]
- 終了時刻: [18:00]
- 出発地点: [ルネッサンス リバーサイド ホテル]
- 移動手段: [Grab（配車アプリ）]
- 予算: [1日あたり{5,000円/10,000円/制限なし}]

【興味のあること】
- 優先度1: [歴史的建造物]
- 優先度2: [ローカルグルメ]
- 優先度3: [市場散策]

【避けたいこと】
- [長時間の徒歩移動]
- [混雑した観光地]

以下の形式で提案してください：
1. 時系列の行動予定（表形式）
2. 各スポットの見どころと所要時間
3. 移動ルートと所要時間
4. ランチ・ディナーの推奨レストラン
5. 費用の内訳
6. 天候や混雑を考慮した代替案
```

**カスタマイズのポイント**:
- 都市名と日付を変更
- 興味の優先順位を明確にする
- 避けたいことを具体的に伝える

**期待される出力**:
- 時間帯ごとの詳細な行動計画
- 移動時間を考慮した現実的なスケジュール
- 代替案を含む柔軟なプラン

### 1-3. 予算最適化プラン

**使用場面**: 限られた予算で最大限の体験を得たい

**プロンプト例**:

```
[ベトナム・タイ8日間]の旅行で、予算を最適化したプランを提案してください。

【予算】
- 総予算: [30万円]
- 内訳の希望:
  - 航空券: [すでに購入済み]
  - 宿泊: [1泊{5,000円/10,000円/15,000円}程度]
  - 食事: [1日{3,000円/5,000円/制限なし}程度]
  - 観光: [1日{3,000円/5,000円/制限なし}程度]
  - その他: [予備費として{2万円/5万円}]

【譲れないこと】
- [清潔で安全なホテル]
- [本格的なローカルグルメ体験]

【節約できること】
- [ホテルのグレード]
- [移動手段（LCCやバス利用OK）]

以下を含めて提案してください：
1. 予算配分の最適案（表形式）
2. 節約ポイントと節約額
3. 費用対効果の高い体験
4. 予算オーバーを防ぐコツ
5. 緊急時の予備費の使い方
```

**カスタマイズのポイント**:
- 総予算と内訳の希望を明確にする
- 譲れないことと節約できることを区別する
- 具体的な金額を示す

**期待される出力**:
- 予算内で実現可能な詳細プラン
- 節約のための具体的なアドバイス
- 予算管理のコツ

## 2. グルメ・レストラン編

### 2-1. レストラン探し（基本）

**使用場面**: 特定の料理やレストランを探す

**プロンプト例**:

```
[ハノイ]で[本格的なフォー]を食べられるレストランを5軒推薦してください。

【条件】
- 予算: [1人{500円以下/1,000円以下/制限なし}]
- 立地: [旧市街から{徒歩圏内/Grabで15分以内}]
- 雰囲気: [{ローカル/観光客向け/高級}]
- 営業時間: [{朝食/ランチ/ディナー}に営業]

【重視すること】
- 優先度1: [味の評価]
- 優先度2: [地元民に人気]
- 優先度3: [清潔さ]

各レストランについて、以下を教えてください：
1. 店名（英語とベトナム語）
2. 住所とGoogleマップのリンク
3. 営業時間
4. 推奨メニューと価格
5. 特徴（雰囲気、客層、混雑状況）
6. 口コミ評価（Google、TripAdvisorなど）
7. 注意点（予約の必要性、支払い方法など）
8. アクセス方法

最後に、5軒を比較した表を作成してください。
```

**カスタマイズのポイント**:
- 都市名と料理名を変更
- 予算と立地の条件を明確にする
- 重視することの優先順位をつける

**期待される出力**:
- 5軒の詳細な情報
- 比較表
- それぞれの特徴と推奨理由

---

### 2-2. 食事制限対応レストラン

**使用場面**: アレルギーや宗教上の理由で食事制限がある

**プロンプト例**:

```
[バンコク]で[ベジタリアン/ヴィーガン/ハラル/グルテンフリー]対応のレストランを探しています。

【食事制限】
- 制限内容: [具体的な制限内容]
- 理由: [アレルギー/宗教/健康/個人の選択]

【希望】
- 料理のジャンル: [タイ料理/和食/洋食/何でもOK]
- 予算: [1人{1,000円以下/2,000円以下/制限なし}]
- 立地: [ホテルから{徒歩圏内/Grabで20分以内}]

【重要な確認事項】
- 完全に対応しているか、部分的に対応しているか
- クロスコンタミネーション（混入）のリスクはあるか
- スタッフは制限内容を理解しているか
- 英語でのコミュニケーションは可能か

各レストランについて、以下を詳しく教えてください：
1. 対応レベル（完全対応/部分対応/要相談）
2. 推奨メニュー
3. 注文時に伝えるべきフレーズ（現地語と英語）
4. 過去の利用者の評価
5. 注意点
```

**カスタマイズのポイント**:
- 制限内容を具体的に伝える
- 完全対応か部分対応かを明確にする
- 言語の壁への対応を確認する

**期待される出力**:
- 制限に対応したレストランリスト
- 注文時に使えるフレーズ
- 安全に食事するためのアドバイス

### 2-3. ローカルグルメ体験

**使用場面**: 観光客向けではない本物のローカル体験をしたい

**プロンプト例**:

```
[ダナン]で地元の人が実際に通う、観光客が少ない本格的なレストランを教えてください。

【条件】
- ローカル度: [観光客比率{30%以下/50%以下}]
- 価格帯: [地元の人が日常的に利用する価格帯]
- 料理: [ダナン名物/ベトナム家庭料理]
- 言語: [英語が通じなくてもOK]

【知りたいこと】
1. 店の見つけ方（目印、外観の特徴）
2. 注文方法（メニューの有無、指差し注文の可否）
3. 支払い方法（現金のみか、カード可か）
4. 地元の人の利用時間帯（混雑を避けるため）
5. 注文すべき定番メニュー
6. マナーや暗黙のルール
7. 一人で行っても大丈夫か
8. 写真撮影は可能か

また、以下のシチュエーションでの対応方法も教えてください：
- メニューが読めない場合
- 注文方法がわからない場合
- 会計方法がわからない場合
```

**カスタマイズのポイント**:
- ローカル度の基準を明確にする
- 言語の壁への対応を確認する
- 文化的なマナーを事前に確認する

**期待される出力**:
- 本物のローカル体験ができる店のリスト
- 言葉が通じなくても注文できる方法
- 文化的なマナーのアドバイス

## 3. 観光・アクティビティ編

### 3-1. 観光スポット選定

**使用場面**: 限られた時間で効率的に観光したい

**プロンプト例**:

```
[チェンマイ]で[2日間]の観光プランを作成してください。

【条件】
- 滞在期間: [8月17日〜18日]
- 興味のあること: [寺院、自然、文化体験]
- 興味のないこと: [ショッピング、ナイトライフ]
- 体力レベル: [{普通/あまり歩けない/アクティブ}]
- 移動手段: [Grab、レンタルバイク{可/不可}]

【重視すること】
- 優先度1: [有名な観光地は外せない]
- 優先度2: [地元の文化に触れたい]
- 優先度3: [写真映えするスポット]

以下を含めて提案してください：
1. 2日間の詳細スケジュール（表形式）
2. 各スポットの見どころと所要時間
3. 入場料や必要な費用
4. 服装や持ち物の注意点
5. 混雑を避ける時間帯
6. 写真撮影のベストスポット
7. 近くのレストランやカフェ
8. 雨天時の代替案
```

**カスタマイズのポイント**:
- 滞在期間と興味を明確にする
- 体力レベルに合わせたプランを依頼する
- 天候への対応を含める

**期待される出力**:
- 効率的な2日間のスケジュール
- 各スポットの詳細情報
- 天候や体調に応じた代替案

```
【重視すること】

3. 料金（含まれるもの、含まれないもの）

6. 持ち物や服装
```

**カスタマイズのポイント**:
- 体験したい内容を具体的に伝える
- 言語対応を確認する
- 予約の必要性を確認する

**期待される出力**:
- 体験型アクティビティのリスト
- 予約方法と注意事項
- 参加者の口コミと評価

## 4. 移動・交通編

### 4-1. 都市間移動の最適化

**使用場面**: 複数の都市を効率的に移動したい

**プロンプト例**:

```
[ホーチミン]から[バンコク]への移動方法を比較してください。

【条件】
- 移動日: [8月16日]
- 出発時刻の希望: [{早朝/午前/午後/夜}]
- 予算: [{最安/標準/快適さ優先}]
- 荷物: [スーツケース1個、機内持ち込みバッグ1個]

【比較してほしいこと】
1. 直行便vs乗り継ぎ便
2. フルサービスキャリアvsLCC
3. 飛行機vs列車vsバス

各移動手段について、以下を教えてください：
1. 所要時間（空港までの移動時間含む）
2. 料金（手荷物料金、座席指定料金含む）
3. 快適さ（座席の広さ、機内サービス）
4. 予約方法と予約のタイミング
5. チェックイン方法
6. 遅延・欠航のリスク
7. メリット・デメリット

最後に、推奨する移動手段とその理由を教えてください。
```

**カスタマイズのポイント**:
- 出発地と目的地を変更
- 予算と快適さの優先順位を明確にする
- 荷物の量を伝える

**期待される出力**:
- 複数の移動手段の詳細比較
- 推奨する移動手段と理由
- 予約方法と注意点

---

### 4-2. 市内交通の使い方

**使用場面**: 現地の交通手段を効率的に使いたい

**プロンプト例**:

```
[ハノイ]での移動手段について教えてください。

【知りたいこと】
1. 主な移動手段（Grab、タクシー、バス、バイクタクシーなど）
2. 各移動手段のメリット・デメリット
3. 料金の目安（空港↔市内、市内の主要スポット間）
4. 安全性と注意点
5. 支払い方法（現金、カード、アプリ）
6. 英語の通じやすさ
7. ぼったくり対策
8. 深夜・早朝の移動方法

特に以下について詳しく教えてください：
- Grabアプリの使い方（登録、支払い設定、配車方法）
- タクシーの見分け方（正規タクシーvs白タク）
- バスの乗り方（路線図、料金、乗降方法）
- 避けるべき移動手段

また、以下のシチュエーションでの対応方法も教えてください：
- 運転手が道を間違えた場合
- 料金でトラブルになった場合
- 荷物が多い場合
```

**カスタマイズのポイント**:
- 都市名を変更
- 特に知りたい移動手段を明確にする
- トラブル対応を含める

**期待される出力**:
- 各移動手段の詳細情報
- Grabなどのアプリの使い方
- トラブル対応のアドバイス

## 5. トラブル対応編

### 5-1. フライトキャンセル・遅延

**使用場面**: フライトがキャンセルまたは大幅に遅延した

**プロンプト例**:

```
[ダナン発バンコク行き]のフライトがキャンセルされました。対応方法を教えてください。

【状況】
```

## 7. プロンプト作成のコツ

### 7-1. 効果的なプロンプトの5原則

1. **具体的に書く**
   - ❌ 「おすすめのレストランを教えて」
   - ✅ 「ハノイ旧市街で、予算1,000円以下、地元民に人気のフォー店を5軒教えて」

2. **文脈を提供する**
   - ❌ 「バンコクの観光プランを作って」
   - ✅ 「バンコクで1日、歴史的建造物に興味があり、徒歩移動は避けたい。9時から18時までの観光プランを作って」

3. **制約条件を明示する**
   - ❌ 「ホテルを探して」
   - ✅ 「ダナンで、予算1泊1万円以下、ビーチから徒歩5分以内、朝食付きのホテルを探して」

4. **出力形式を指定する**
   - ❌ 「移動方法を比較して」
   - ✅ 「飛行機、列車、バスの3つを、料金・所要時間・快適さで比較した表を作って」

5. **優先順位をつける**
   - ❌ 「清潔で安くて便利なホテル」
   - ✅ 「優先度1: 清潔さ、優先度2: 立地、優先度3: 価格のホテル」

### 7-2. よくある失敗とその改善

| 失敗例 | 問題点 | 改善例 |
|:---|:---|:---|
| 「いい感じのレストラン」 | 曖昧すぎる | 「地元民に人気で、予算1,000円以下、清潔なレストラン」 |
| 「観光プランを作って」 | 情報不足 | 「1日、歴史重視、徒歩少なめ、9-18時の観光プラン」 |
| 「安いホテル」 | 基準が不明確 | 「1泊5,000円以下のホテル」 |
| 「おすすめを教えて」 | 主観的すぎる | 「口コミ評価4.0以上、日本人利用者の多いホテル」 |

### 7-3. プロンプトのテンプレート構造

効果的なプロンプトは、以下の構造を持っています：

```
【依頼内容】
何をしてほしいかを1文で明確に書く

【条件】
- 必須条件（絶対に満たすべきこと）
- 希望条件（できれば満たしてほしいこと）
- 制約条件（避けたいこと、できないこと）

【優先順位】
- 優先度1: 最も重要なこと
- 優先度2: 次に重要なこと
- 優先度3: できれば考慮してほしいこと

【出力形式】
- 表形式、リスト形式、段落形式など
- 含めてほしい情報の項目

【追加の質問】
- 特に詳しく知りたいこと
- 想定されるシチュエーションへの対応
```
