## レビューの観点

### 1. 構成・整合性（最優先）
- [ ] 章の順序や節の番号に矛盾や錯乱がないか
- [ ] 「次章への案内」と実際の次章タイトルが一致しているか
- [ ] 引用ブロック（`> `で始まる部分）内に章節番号付き見出しがないか
- [ ] 同じ内容が重複して掲載されていないか
- [ ] 参考ログ表に重複や誤りがないか

### 2. 誤字・脱字・表記ゆれ
- [ ] 明らかな誤字・脱字がないか
- [ ] 表記ゆれ（例：「コロン」が半角と全角で混在）がないか
- [ ] 一人称の統一（「私」「著者」の使い分けが適切か）
- [ ] 数字・記号の全角/半角が統一されているか

### 3. 事実関係の正確性
- [ ] 地名・施設名・サービス名が正確か
- [ ] AIの回答例に明らかな誤情報がないか（例：存在しない駅名、誤った料金情報）
- [ ] 日付・時刻の記載に矛盾がないか

### 4. 文章の品質
- [ ] 読みやすく、分かりやすい文章か
- [ ] 冗長な表現や重複がないか
- [ ] 専門用語に適切な説明があるか

### 5. ドラフト文言の残存
- [ ] 「このドラフトは...」「進捗：XX%」などの執筆メモが残っていないか
- [ ] 「TODO」「要確認」などのマーカーが残っていないか
