# 付録B：プロンプトテンプレート集

## 本付録について

本付録は、旅行でChatGPTを活用する際に、読者がすぐに使えるプロンプトテンプレート集です。旅行の各フェーズ（計画、現地での情報収集、トラブル対応、振り返り）で使える実践的なテンプレートを、カテゴリ別に整理して掲載しています。

### テンプレートの構造

各テンプレートは以下の構造になっています：

- **プロンプト本文**：`[  ]` で囲まれた箇所を、あなたの状況に合わせて置き換えてください
- **期待される出力**：ChatGPTからどのような回答が得られるかの例
- **カスタマイズのポイント**：より良い結果を得るためのヒント

### 使い方

1. **コピー&ペースト**：テンプレートをそのままChatGPTにコピー&ペーストします
2. **カスタマイズ**：`[  ]` の箇所を、あなたの状況に合わせて書き換えます
3. **複数回の対話**：最初の回答をもとに、さらに質問を重ねることで、より詳細な情報が得られます

それでは、カテゴリ別にテンプレートを見ていきましょう。

---

## 1. 旅行計画編

旅行の計画段階で使えるテンプレートです。旅程作成、観光プラン、予算最適化など、出発前の準備に役立ちます。

### テンプレート1：基本的な旅程作成

```
[目的地]で[日数]日間の旅行を計画しています。以下の条件で旅程を作成してください：

- 旅行期間：[出発日]〜[帰国日]
- 人数：[人数]
- 予算：[予算]
- 興味のあること：[興味のあること]
- 避けたいこと：[避けたいこと]

各日の詳細なスケジュール、移動手段、宿泊先の候補を含めてください。
```

**期待される出力**：日ごとの詳細なスケジュール、移動手段、宿泊先の候補、概算費用

**カスタマイズのポイント**：興味のあることや避けたいことを具体的に記載すると、よりあなたに合ったプランが提案されます。例えば「歴史的な建造物を見たい」「混雑した観光地は避けたい」など。

---

### テンプレート2：特定の都市の観光プラン

```
[都市名]で[日数]日間滞在します。以下の条件で観光プランを作成してください：

- 滞在期間：[出発日]〜[帰国日]
- 宿泊エリア：[エリア名]
- 移動手段：[徒歩/公共交通/タクシー/レンタカー]
- 興味のある観光スポット：[観光スポット]
- 食事の好み：[食事の好み]

各日の詳細なスケジュール、移動時間、食事の候補を含めてください。
```

**期待される出力**：日ごとの詳細なスケジュール、各スポット間の移動時間、おすすめのレストラン

**カスタマイズのポイント**：宿泊エリアと移動手段を明記すると、より実現可能なプランが得られます。例えば「旧市街に宿泊、基本は徒歩で移動」など。

---

### テンプレート3：予算最適化プラン

```
[目的地]で[日数]日間の旅行を計画しています。予算は[予算]です。以下の項目について、コストを抑えるためのアドバイスをください：

- 航空券：[出発地]→[目的地]、[出発日]〜[帰国日]
- 宿泊：[希望する宿泊タイプ]
- 食事：[1日あたりの食費予算]
- 観光：[訪れたい観光スポット]
- 交通：[移動手段の希望]

予算内で最大限楽しむための具体的な提案をお願いします。
```

**期待される出力**：各項目のコスト削減方法、予算配分の提案、代替案、お得な予約サイトの紹介

**カスタマイズのポイント**：予算を具体的に記載し、優先順位を明確にすると、より実用的なアドバイスが得られます。例えば「宿泊は節約したいが、食事は現地の美味しいものを食べたい」など。

---

## 2. グルメ・レストラン編

現地での食事を楽しむためのテンプレートです。レストラン探し、食事制限対応、ローカルグルメ体験に役立ちます。

### テンプレート1：レストラン探し（基本）

```
[都市名]の[エリア名]で、以下の条件に合うレストランを探しています：

- 料理のジャンル：[料理のジャンル]
- 予算：[1人あたりの予算]
- 雰囲気：[希望する雰囲気]
- 営業時間：[希望する時間帯]
- その他の希望：[その他の希望]

おすすめのレストランを3〜5件、理由とともに教えてください。
```

**期待される出力**：レストラン名、住所、予算、おすすめ理由、営業時間、予約の必要性

**カスタマイズのポイント**：雰囲気や営業時間を具体的に記載すると、より適切な提案が得られます。例えば「静かで落ち着いた雰囲気」「ランチタイムに営業している店」など。

---

### テンプレート2：食事制限対応レストラン

```
[都市名]で、以下の食事制限に対応したレストランを探しています：

- 食事制限：[ベジタリアン/ビーガン/グルテンフリー/ハラール/アレルギー]
- 料理のジャンル：[料理のジャンル]
- 予算：[1人あたりの予算]
- エリア：[エリア名]

食事制限に対応していて、美味しいと評判のレストランを教えてください。
```

**期待される出力**：レストラン名、対応している食事制限の詳細、メニュー例、評判、予約方法

**カスタマイズのポイント**：食事制限を具体的に記載し、アレルギーがある場合は詳細を伝えると、より安全な店が提案されます。例えば「ナッツアレルギーあり、完全に分離された調理環境が必要」など。

---

### テンプレート3：ローカルグルメ体験

```
[都市名]で、地元の人が通うようなローカルグルメを体験したいです。以下の条件で教えてください：

- 食べたい料理：[料理名]
- 予算：[1人あたりの予算]
- エリア：[エリア名]
- 雰囲気：[観光客向けではなく、地元の人が通う店]

地元の人に人気の店を教えてください。注文方法や食べ方のコツも含めて。
```

**期待される出力**：店名、住所、おすすめメニュー、注文方法、食べ方のコツ、現地語での注文フレーズ

**カスタマイズのポイント**：「観光客向けではなく」と明記すると、よりローカルな店が提案されます。また「英語メニューがなくても大丈夫」と伝えると、さらにディープな店が見つかります。

---

## 3. 観光・アクティビティ編

現地での観光やアクティビティを楽しむためのテンプレートです。観光スポット選定、体験型アクティビティ、穴場スポット探しに役立ちます。

### テンプレート1：観光スポット選定

```
[都市名]で[日数]日間滞在します。以下の条件で観光スポットを選定してください：

- 興味のあること：[歴史/文化/自然/アート/建築/グルメ]
- 移動手段：[徒歩/公共交通/タクシー]
- 宿泊エリア：[エリア名]
- 避けたいこと：[混雑/長時間の移動/高額な入場料]

おすすめの観光スポットを5〜10件、理由とともに教えてください。
```

**期待される出力**：観光スポット名、住所、入場料、所要時間、おすすめ理由、混雑状況

**カスタマイズのポイント**：興味のあることと避けたいことを具体的に記載すると、よりあなたに合ったスポットが提案されます。例えば「美術館が好きだが、2時間以上かかる場所は避けたい」など。

---

### テンプレート2：体験型アクティビティ

```
[都市名]で、以下の条件に合う体験型アクティビティを探しています：

- アクティビティのタイプ：[料理教室/伝統工芸/スポーツ/自然体験]
- 予算：[1人あたりの予算]
- 所要時間：[希望する時間]
- 言語：[日本語/英語/現地語]

おすすめのアクティビティを3〜5件、理由とともに教えてください。
```

**期待される出力**：アクティビティ名、内容の詳細、予算、所要時間、予約方法、言語対応

**カスタマイズのポイント**：言語対応を明記すると、より適切な提案が得られます。また「初心者でも参加可能」「子供連れでも大丈夫」などの条件を追加すると、さらに絞り込めます。

---

### テンプレート3：穴場スポット探し

```
[都市名]で、観光客があまり訪れない穴場スポットを探しています：

- 興味のあること：[興味のあること]
- エリア：[エリア名、または「どこでも」]
- 移動手段：[徒歩/公共交通/タクシー]

地元の人が知っている穴場スポットを教えてください。
```

**期待される出力**：穴場スポット名、住所、特徴、行き方、なぜ穴場なのか

**カスタマイズのポイント**：「観光客があまり訪れない」と明記すると、よりローカルなスポットが提案されます。また「ガイドブックに載っていない」と追加すると、さらにディープな場所が見つかります。

---

## 4. 移動・交通編

現地での移動をスムーズにするためのテンプレートです。都市間移動、市内交通、空港アクセスに役立ちます。

### テンプレート1：都市間移動の最適化

```
[出発地]から[目的地]への移動方法を教えてください：

- 出発日時：[出発日時]
- 人数：[人数]
- 荷物：[大きな荷物の有無]
- 予算：[予算]
- 優先順位：[速さ/安さ/快適さ]

おすすめの移動方法を3つ、理由とともに教えてください。
```

**期待される出力**：移動方法（飛行機/鉄道/バス/タクシー）、所要時間、料金、予約方法、メリット・デメリット

**カスタマイズのポイント**：優先順位を明確にすると、より適切な提案が得られます。例えば「時間を節約したいので、多少高くても飛行機が良い」「景色を楽しみたいので鉄道が良い」など。

---

### テンプレート2：市内交通の使い方

```
[都市名]での市内交通について教えてください：

- 滞在期間：[日数]
- 主な移動先：[移動先]
- 予算：[1日あたりの交通費予算]

おすすめの交通手段（地下鉄/バス/タクシー/配車アプリ）と、お得な乗車券があれば教えてください。
```

**期待される出力**：交通手段の種類、料金、お得な乗車券（1日券、3日券など）、使い方のコツ、アプリの紹介

**カスタマイズのポイント**：主な移動先を記載すると、より具体的な提案が得られます。例えば「旧市街から空港まで毎日往復する」など。

---

### テンプレート3：空港アクセス

```
[空港名]から[目的地]への移動方法を教えてください：

- 到着時刻：[到着時刻]
- 人数：[人数]
- 荷物：[大きな荷物の有無]
- 予算：[予算]

おすすめの移動方法を3つ、理由とともに教えてください。
```

**期待される出力**：移動方法（鉄道/バス/タクシー/配車アプリ）、所要時間、料金、乗り場の場所、深夜・早朝の対応

**カスタマイズのポイント**：到着時刻を記載すると、深夜や早朝の移動にも対応した提案が得られます。また「大きなスーツケース2個あり」と伝えると、荷物の積載を考慮した提案がされます。

---

## 5. トラブル対応編

旅行中のトラブルに対応するためのテンプレートです。フライトキャンセル、体調不良、盗難・紛失などの緊急時に役立ちます。

### テンプレート1：フライトキャンセル・遅延

```
[航空会社]の[便名]がキャンセル/遅延しました。以下の状況で対応方法を教えてください：

- 現在地：[現在地]
- 目的地：[目的地]
- 出発予定時刻：[出発予定時刻]
- 次の予定：[次の予定]

航空会社への連絡方法、代替便の探し方、補償の受け方を教えてください。
```

**期待される出力**：航空会社への連絡方法、代替便の探し方、補償の受け方、次のステップ、緊急連絡先

**カスタマイズのポイント**：次の予定を記載すると、より具体的な対応方法が提案されます。例えば「3時間後にホテルのチェックインがある」「夜には次の都市に移動する予定」など。

---

### テンプレート2：体調不良・病気

```
[都市名]で体調不良になりました。以下の症状で対応方法を教えてください：

- 症状：[症状]
- 現在地：[現在地]
- 言語：[日本語/英語/現地語]
- 海外旅行保険：[加入している/していない]

病院の探し方、診察の受け方、保険の使い方を教えてください。
```

**期待される出力**：病院の探し方、診察の受け方、保険の使い方、緊急連絡先、現地語での症状の伝え方

**カスタマイズのポイント**：症状を具体的に記載し、海外旅行保険の加入状況を伝えると、より適切なアドバイスが得られます。また「英語が話せる病院が良い」と追加すると、言語対応を考慮した提案がされます。

---

### テンプレート3：盗難・紛失

```
[都市名]で[盗まれた/紛失した物]を[盗まれました/紛失しました]。以下の状況で対応方法を教えてください：

- 盗まれた/紛失した物：[物]
- 場所：[場所]
- 時刻：[時刻]
- 現在地：[現在地]

警察への届け出、大使館への連絡、再発行の手続きを教えてください。
```

**期待される出力**：警察への届け出方法、大使館への連絡方法、再発行の手続き、次のステップ、緊急連絡先

**カスタマイズのポイント**：盗まれた/紛失した物を具体的に記載すると、より適切な対応方法が提案されます。特に「パスポート」「クレジットカード」などの重要な物の場合、緊急の手続きが必要です。

## 6. 帰国後・振り返り編

旅行後の記録整理や次回の計画に役立つテンプレートです。旅行記作成、費用分析、次回の計画に活用できます。

### テンプレート1：旅行記録の整理

```
[目的地]での[日数]日間の旅行を振り返りたいです。以下の情報から、旅行記を作成してください：

- 訪れた場所：[場所のリスト]
- 食べた料理：[料理のリスト]
- 印象的だったこと：[印象的だったこと]
- 写真：[写真の説明]

ブログやSNSに投稿できる形式で、旅行記を作成してください。
```

**期待される出力**：旅行記（導入→各日の出来事→まとめ）、ハッシュタグ、写真の配置案、読者を引き込む文章

**カスタマイズのポイント**：印象的だったことを具体的に記載すると、より魅力的な旅行記が作成されます。また「Instagram用に短くまとめて」「ブログ用に詳しく書いて」など、用途を明記すると、適切な長さで作成されます。

---

### テンプレート2：費用の分析

```
[目的地]での[日数]日間の旅行費用を分析したいです。以下の情報から、費用の内訳と節約できた点を教えてください：

- 航空券：[金額]
- 宿泊：[金額]
- 食事：[金額]
- 観光：[金額]
- 交通：[金額]
- その他：[金額]

合計金額、各項目の割合、節約できた点、次回への改善点を教えてください。
```

**期待される出力**：合計金額、各項目の割合（円グラフのデータ）、節約できた点、次回への改善点、同じ目的地の平均的な費用との比較

**カスタマイズのポイント**：各項目の金額を正確に記載すると、より詳細な分析が得られます。また「次回はもっと節約したい」「次回は食事にもっとお金をかけたい」など、目標を伝えると、具体的なアドバイスがもらえます。

---

### テンプレート3：次回の旅行計画

```
今回の[目的地]での旅行を踏まえて、次回の旅行を計画したいです。以下の情報から、おすすめの目的地を教えてください：

- 今回の旅行で良かった点：[良かった点]
- 今回の旅行で改善したい点：[改善したい点]
- 次回の旅行で体験したいこと：[体験したいこと]
- 予算：[予算]
- 時期：[時期]

おすすめの目的地を3〜5件、理由とともに教えてください。
```

**期待される出力**：おすすめの目的地、理由、旅行のテーマ、概算費用、ベストシーズン

**カスタマイズのポイント**：今回の旅行で良かった点と改善したい点を具体的に記載すると、よりあなたに合った目的地が提案されます。例えば「今回は都市観光が中心だったので、次回は自然を楽しみたい」など。

## 7. プロンプト作成のコツ

効果的なプロンプトを作成するための5つの原則と、よくある失敗例を紹介します。

### 効果的なプロンプトの5原則

#### 1. 具体的に

曖昧な表現を避け、具体的な情報を提供しましょう。

**悪い例**：「ハノイでおすすめのレストランを教えて」  
**良い例**：「ハノイ旧市街で、予算1,000円以下、フォーが美味しい店を3件教えて」

#### 2. 文脈を提供

あなたの状況や背景を説明しましょう。

**悪い例**：「バンコクの観光スポットを教えて」  
**良い例**：「バンコクに3日間滞在します。歴史的な寺院に興味があり、宿泊先はスクンビットエリアです。おすすめの観光スポットを教えて」

#### 3. 優先順位を明確に

何を重視するかを明記しましょう。

**悪い例**：「ハノイからハロン湾への移動方法を教えて」  
**良い例**：「ハノイからハロン湾への移動方法を教えて。時間を節約したいので、多少高くても速い方法が良いです」

#### 4. 制約条件を示す

予算、時間、避けたいことを伝えましょう。

**悪い例**：「ダナンでアクティビティを探しています」  
**良い例**：「ダナンでアクティビティを探しています。予算は5,000円以内、所要時間は半日以内、激しいスポーツは避けたいです」

#### 5. 期待する出力を示す

どのような形式で回答してほしいかを明記しましょう。

**悪い例**：「ベトナム旅行の計画を立てて」  
**良い例**：「ベトナム旅行の7日間の計画を立ててください。各日のスケジュール、移動手段、宿泊先の候補を表形式で教えて」

---

### よくある失敗とその改善

| 失敗例 | 問題点 | 改善例 |
|:---|:---|:---|
| 「おすすめのレストランを教えて」 | 情報が少なすぎる | 「ハノイ旧市街で、予算1,000円以下、フォーが美味しい店を教えて」 |

## 8. 付録の活用例

本付録のテンプレートを、実際の旅行でどのように活用できるかを、3つのシナリオで紹介します。

### シナリオ1：初めての海外旅行

**状況**：初めてベトナムに行く。英語も現地語も話せない。不安が多い。

**活用するテンプレート**：
1. **旅行計画編 - テンプレート1**：基本的な旅程を作成
2. **グルメ・レストラン編 - テンプレート1**：レストラン探し
3. **移動・交通編 - テンプレート2**：市内交通の使い方
4. **トラブル対応編 - テンプレート1〜3**：緊急時の対応方法を事前に確認

**ポイント**：出発前に、トラブル対応のテンプレートを使って、緊急時の対応方法を確認しておくと安心です。

---

### シナリオ2：短期間の弾丸旅行

**状況**：3日間でバンコクとチェンマイを回る。時間が限られている。

**活用するテンプレート**：
1. **旅行計画編 - テンプレート2**：各都市の効率的な観光プラン
2. **移動・交通編 - テンプレート1**：都市間移動の最適化
3. **観光・アクティビティ編 - テンプレート1**：短時間で回れる観光スポット

**ポイント**：「移動時間を最小限にしたい」「各スポットの所要時間を教えて」など、時間を重視したプロンプトを使いましょう。

---

### シナリオ3：長期滞在

**状況**：1ヶ月間ベトナムに滞在。ゆっくり現地の生活を楽しみたい。

**活用するテンプレート**：
1. **グルメ・レストラン編 - テンプレート3**：ローカルグルメ体験
2. **観光・アクティビティ編 - テンプレート2・3**：体験型アクティビティ、穴場スポット
3. **帰国後・振り返り編 - テンプレート3**：次回の旅行計画

**ポイント**：「観光客向けではなく」「地元の人が通う」など、ローカルな体験を重視したプロンプトを使いましょう。
