# 対話型実例の候補選定メモ

## 目的
第3章の各節で1つずつ、AIとのキャッチボールが面白い対話型実例を作成する。

## 選定基準
1. **対話の往復が複数回ある**（3〜5往復程度）
2. **思考プロセスが見える**（試行錯誤、条件の絞り込み、代替案の検討など）
3. **物語性がある**（読者が「自分も同じように悩むだろう」と共感できる）
4. **実用的な学びがある**（読者が真似できる対話パターン）

## 候補ログの分析

### 第1節：旅行プランの立案

#### 候補1：初期プラン提案（LOG001）
**ファイル**: `2025-7-25 22-18-43-_______.md`（27K）
**内容**: 
- 著者「行きと帰りの飛行機を決めてますが、他決めてません」
- AI「訪れたい都市、目的、ホテル予算などを教えてください」
- 著者「ホイアンが近ければ行きたい。基本的にホーチミンとバンコクで。観光と仕事で」
- AI「3つのプラン案を提案します」

**評価**: ⭐⭐⭐⭐⭐
- 対話は2往復だが、AIが質問を投げかけ、著者が条件を絞り込むプロセスが明確
- 「ホイアンが近ければ」という曖昧な希望をAIがどう処理するかが見える
- 3つのプラン案が提示され、選択肢の提示方法が学べる

**対話型実例としての展開案**:
1. 著者「航空券以外何も決まっていない」
2. AI「追加情報を教えてください」（質問リスト）
3. 著者「ホイアンに行きたいが、基本はホーチミンとバンコク」
4. AI「ホイアンはホーチミンから遠いですが、ダナン経由で可能です。ただし3日必要で、バンコクの日数が減ります」
5. 著者「それなら今回は見送って、バンコクを充実させよう」
6. AI「了解しました。3つのプラン案を提案します」
