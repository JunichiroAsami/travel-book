## 1. 付録の概要

### 1.1 付録のテーマ

本付録は、旅行でChatGPTを活用する際に、読者がすぐに使えるプロンプトテンプレート集を提供する。旅行の各フェーズ（計画、現地での情報収集、トラブル対応、振り返り）で使える実践的なテンプレートを、カテゴリ別に整理して掲載する。

### 1.2 付録の役割

- **即戦力の提供**: 読者がコピー&ペーストですぐに使えるテンプレート
- **学習素材の提供**: 効果的なプロンプトの書き方を学べる
- **カスタマイズの基礎**: テンプレートをベースに、自分の状況に合わせてカスタマイズできる
- **実用性の確保**: 本文の理論を実践に移すための具体的なツール

### 1.3 提供方法

**本書の末尾に収録**

付録Bは、読者が購入後すぐに使える実用的なテンプレート集として、本書の末尾に収録します。

**理由**:
- 即戦力として、購入後すぐに使える
- 本の価値を高める（実用性の確保）
- セミナーへの誘導（「もっと詳しく知りたい人はセミナーへ」）

## 2. 付録の構成

### 2.1 構成の概要

本付録は、以下の構成でプロンプトテンプレートを提供する：

1. **導入部**: 本付録について、使い方、テンプレートの構造
2. **旅行計画編**: 旅程作成、観光プラン、予算最適化
3. **グルメ・レストラン編**: レストラン探し、食事制限対応、ローカルグルメ
4. **観光・アクティビティ編**: 観光スポット選定、体験型アクティビティ、穴場スポット
5. **移動・交通編**: 都市間移動、市内交通、空港アクセス
6. **トラブル対応編**: フライトキャンセル、体調不良、盗難・紛失

8. **プロンプト作成のコツ**: 効果的なプロンプトの5原則、よくある失敗
9. **付録の活用例**: 初めての海外旅行、短期間の弾丸旅行、長期滞在

#### 第8節：プロンプト作成のコツ

**テーマ**: 効果的なプロンプトの5原則、よくある失敗

**内容**:

**効果的なプロンプトの5原則**

1. **具体的に**: 曖昧な表現を避け、具体的な情報を提供する
2. **文脈を提供**: 状況や背景を説明する
3. **優先順位を明確に**: 何を重視するかを明記する
4. **制約条件を示す**: 予算、時間、避けたいことを伝える
5. **期待する出力を示す**: どのような形式で回答してほしいかを明記する

**よくある失敗とその改善**

| 失敗例 | 改善例 |
|:---|:---|
| 「おすすめのレストランを教えて」 | 「ハノイ旧市街で、予算1,000円以下、フォーが美味しい店を教えて」 |
| 「旅行計画を作って」 | 「ベトナムで7日間、予算10万円、歴史と食文化に興味がある旅行計画を作って」 |
| 「移動方法を教えて」 | 「ハノイからハロン湾へ、2人、大きな荷物あり、予算5,000円以内の移動方法を教えて」 |

**プロンプトのテンプレート構造**

```
[状況の説明] + [具体的な条件] + [期待する出力]
```
