# 付録A執筆コスト見積もり：MANUS vs Gemini

## 前提条件

### 付録Aの仕様
- **目標文字数**: 70,000〜140,000文字
- **構成**: 導入部 + 生ログ20件（各3,000〜7,000文字）+ まとめ
- **作業内容**:
  1. giant_material_databaseから主要ログ20件を選定
  2. 各ログに4つの解説を追加（1,000〜2,000文字 × 4 = 4,000〜8,000文字/件）
  3. 導入部とまとめの執筆

---

## オプション1: MANUSで執筆

### 作業の内訳

#### Phase 1: ログの選定と抽出
- giant_material_database.mdから主要ログ20件を選定
- 各ログの全文を抽出
- **推定トークン数**: 約5,000トークン（選定・抽出作業）

#### Phase 2: 導入部の執筆
- 約2,000文字
- **推定トークン数**: 約3,000トークン

#### Phase 3: 各ログの解説執筆（20件）
- 1件あたり約4,000〜8,000文字の解説
- 20件 × 平均6,000文字 = 120,000文字
- **推定トークン数**: 約180,000トークン（20件 × 9,000トークン/件）

#### Phase 4: まとめの執筆
- 約1,000〜2,000文字
- **推定トークン数**: 約2,000トークン

#### Phase 5: レビューと修正
- 全体のレビュー
- 修正作業
- **推定トークン数**: 約30,000トークン

### 合計推定トークン数

| フェーズ | トークン数 |
|:---|---:|
| ログの選定と抽出 | 5,000 |
| 導入部の執筆 | 3,000 |
| 各ログの解説執筆（20件） | 180,000 |
| まとめの執筆 | 2,000 |
| レビューと修正 | 30,000 |
| **合計** | **220,000** |

### コスト計算

**注意**: MANUSの料金体系は公開されていないため、正確なコストは不明です。ただし、一般的なAIサービスの料金体系を参考にすると：

- **仮定**: 1,000トークンあたり約0.5〜2円（モデルにより異なる）
- **推定コスト**: 220,000トークン × (0.5〜2円 / 1,000トークン) = **約110〜440円**

**ただし**: 実際のコストはMANUSの料金プランによります。

### 所要時間
- **推定時間**: 約2〜4時間（解説の質により変動）

## オプション2: Geminiで執筆（マニュアル操作）

### 作業の内訳

#### Phase 1: プロンプトの準備
- 既に完了
- **コスト**: 0円

#### Phase 2: Geminiへの依頼
- プロンプト + 詳細設計書 + giant_material_database.mdを入力
- Geminiがドラフトを生成
- **コスト**: Gemini 1.5 Proの無料版を使用する場合は0円

#### Phase 3: ドラフトの取得と保存
- 生成されたドラフトをコピー&ペースト
- **コスト**: 0円

#### Phase 4: レビューと修正（MANUSで実施）
- MANUSでレビューと修正
- **推定トークン数**: 約30,000〜50,000トークン
- **推定コスト**: 約15〜100円

### 合計コスト
- **Gemini（無料版）**: 0円
- **MANUSでのレビュー・修正**: 約15〜100円
- **合計**: **約15〜100円**

### 所要時間
- **Geminiでの生成**: 約20〜35分
- **レビューと修正**: 約1〜2時間
- **合計**: 約1.5〜2.5時間

## 推奨

### 第1推奨: Gemini（無料版） + MANUSでレビュー・修正

**理由**:
1. **コストが最も低い**: 約15〜100円
2. **品質も十分**: Geminiが初稿を作成し、MANUSでレビュー・修正することで高品質を確保
3. **所要時間も許容範囲**: 約1.5〜2.5時間

### 第2推奨: MANUSで全て執筆

**理由**:
1. **品質が最も高い**: MANUSが一貫して執筆するため、統一感がある
2. **手間が少ない**: マニュアル操作不要
3. **コストも許容範囲**: 約110〜440円（仮定）
