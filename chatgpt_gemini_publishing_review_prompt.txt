## レビューの目的

この原稿が以下の3つの基準を満たしているかを評価してください：

1. **出版可能レベルの品質**：技術的な問題（構成の破綻、誤字脱字、論理的矛盾など）がなく、そのまま入稿できる状態か
2. **商業的な魅力**：書店に並んだときに売れそうか、読者にとって価値があるか
3. **読者体験**：違和感なく読めるか、最後まで読み通せるか

## レビュー観点

### 1. 出版可能レベルの品質チェック

以下の観点で、原稿の品質を厳しくチェックしてください：

#### 構成・整合性
- [ ] 目次と本文の内容が一致しているか
- [ ] 章の順序が論理的か、流れが自然か
- [ ] 「はじめに」「おわりに」と本文の内容が整合しているか
- [ ] 章と章のつながりが自然か（「次章への案内」が正確か）
- [ ] 節番号が正しく振られているか（8.1.1 → 8.1.2 → 8.2.1 のような順序）

#### 文章・表現
- [ ] 誤字脱字がないか
- [ ] 表記ゆれがないか（「AI」「ＡＩ」「人工知能」など）
- [ ] 一人称が統一されているか（「私」「著者」「筆者」など）
- [ ] 専門用語の使い方が適切か
- [ ] 文体が統一されているか（です・ます調、だ・である調）

#### 付録・参考資料
- [ ] 付録の構成が適切か
- [ ] 付録の使い方ガイドが分かりやすいか
- [ ] プロンプトテンプレートが実用的か
- [ ] ログの引用が適切か

#### 技術的な問題
- [ ] Markdown記法が正しく使われているか
- [ ] 見出しレベルが適切か（H1、H2、H3の使い分け）
- [ ] コードブロックが適切に使われているか
- [ ] 表組みが正しく表示されるか

### 2. 商業的な魅力

以下の観点で、この本が売れそうかを評価してください：

#### ターゲット読者
- [ ] ターゲット読者が明確か
- [ ] ターゲット読者のニーズに応えているか
- [ ] 読者が「自分のために書かれた本だ」と感じられるか

#### 独自性・差別化
- [ ] 他のAI本や旅行記と差別化できているか
- [ ] 「この本でしか得られない価値」が明確か
- [ ] 実体験に基づく具体性があるか

#### 実用性
- [ ] すぐに使えるプロンプトテンプレートがあるか
- [ ] 読者が実際に旅行で活用できる内容か
- [ ] ビジネスや日常生活にも応用できるか

#### タイトル・コンセプト
- [ ] タイトルが魅力的か
- [ ] 副題が内容を適切に表現しているか
- [ ] コンセプトが明確か

#### ボリューム・価格設定
- [ ] 文字数が適切か（多すぎず、少なすぎず）
- [ ] 価格設定が妥当か（推定価格を提示してください）

### 3. 読者体験

以下の観点で、読者が違和感なく読めるかを評価してください：

#### 読みやすさ
- [ ] 文章が平易で分かりやすいか
- [ ] 専門用語が適切に説明されているか
- [ ] 段落の長さが適切か

#### ストーリー性
- [ ] 旅行記としてのストーリー性があるか
- [ ] 著者の体験が生き生きと描かれているか
- [ ] 読者が追体験できるか

#### 実用性とエンタメ性のバランス
- [ ] 実用書としての価値があるか
- [ ] 読み物としての面白さがあるか
- [ ] バランスが取れているか

#### 情報の信頼性
- [ ] 事実関係が正確か
- [ ] 情報源が明示されているか
- [ ] AIの回答をそのまま鵜呑みにしていないか

## レビュー形式

以下の形式でレビュー結果を報告してください：

### 1. 総合評価

- **出版可否判定**：Yes / No / 条件付きYes
- **推定販売部数**：初版〇〇部、累計〇〇部（根拠も記載）
- **推定価格**：〇〇円（根拠も記載）
- **総合評価**：★★★★★（5段階評価）

### 2. 強み（最低3つ）

この原稿の優れている点を具体的に挙げてください。

### 3. 弱み（最低3つ）

この原稿の改善すべき点を具体的に挙げてください。

### 4. 致命的な問題（あれば）

出版前に必ず修正すべき問題を、優先度順に列挙してください。

- **優先度**：★★★★★（最優先）～★☆☆☆☆（低優先）
- **問題箇所**：具体的な章・節・行番号
- **問題内容**：何が問題か
- **修正案**：どう修正すべきか

### 5. 改善提案（任意）

出版の成功確率を高めるための改善提案を、具体的に挙げてください。

### 6. 市場分析

- **競合書籍**：類似書籍との比較
- **市場ポジション**：この本がどのような位置づけになるか
- **販売戦略**：どのような販売戦略が有効か

### 7. 編集者としての最終コメント

この原稿を出版すべきか、その理由を率直に述べてください。
