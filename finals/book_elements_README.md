## Word原稿への統合方法

### 統合順序

1. **扉**: 1ページ目に画像として挿入
2. **目次**: 2ページ目以降（既存の目次を修正）
3. **本文**: 第1章から第5章まで
4. **ディスクレーマー**: 本文の後に挿入
5. **著者プロフィール**: ディスクレーマーの後に挿入（オプション）
6. **奥付**: 最終ページに挿入

### 統合手順

1. 既存のWord原稿（`complete_manuscript_v27_safe.docx`）を開く
2. タイトルを新しいものに変更
3. 1ページ目に扉画像を挿入
4. 最終ページにディスクレーマーと奥付を追加
5. 新しいファイル名で保存（`complete_manuscript_v28.docx`）
