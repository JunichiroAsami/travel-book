# 第1章 素材ファイル：3つの重要エピソード

## 1. スーツケース破損 → AIで保険請求 → 45,300円獲得

- **状況**: ベトナムからの帰国後、スーツケース（Samsonite）のキャスターが破損しているのを発見。
- **AIとの協業**: 
    1. **保険適用の確認**: 住信SBIネット銀行のプラチナデビットカード付帯保険が使えるかAIに相談。
    2. **必要書類の特定**: AIが「携行品損害事故内容報告書」「修理不能証明書」「写真」などが必要と特定。
    3. **書類作成の支援**: AIが「事故内容報告書」の具体的な書き方を提示。
    4. **保険会社とのやり取り**: AIが作成した文面を元に保険会社とやり取り。
- **結果**: 修理費用48,300円から免責3,000円を引いた**45,300円**の保険金が支払われた。
- **ログ**: giant_material_database.md (line: 36439-36584)

## 2. 差し歯が取れた → AIで緊急歯科受診

- **状況**: ベトナム・ホーチミン滞在中に、食事中に奥歯の銀歯と前歯のインプラント仮歯が同時に脱落。
- **AIとの協業**:
    1. **状況分析と応急処置**: AIに状況を伝え、痛みがないこと、審美的な問題が大きいことを整理。応急処置として、取れた歯をなくさないように指示を受ける。
    2. **歯科医院の探索**: AIがホーチミン市内で日本人対応可能、かつ評判の良い歯科医院「ありが歯科」「DYMメディカルセンター」をリストアップ。
    3. **保険適用の検討**: AIが住信SBI、楽天ゴールド、三井住友ゴールドのカード付帯保険の規約を調査。「外傷でない歯科治療」は対象外の可能性が高いが、「応急処置」として請求できる可能性を指摘。
    4. **予約と症状説明**: AIが作成した英語の症状説明文を使い、DYMメディカルセンターに予約。
- **結果**: DYMメディカルセンターで応急処置（仮の接着）をしてもらい、無事に帰国。治療費は保険でカバーされた（約25,000円）。
- **ログ**: giant_material_database.md (line: 9625-11452, 35803-36663)

## 3. 橋の通行料で不当請求 → AIで撃退

- **状況**: タイ・バンコクの運河にかかる小さな橋を渡ろうとした際、現地人に「通行料だ」と20バーツを請求される。
- **AIとの協業**:
    1. **リアルタイム相談**: すぐにAIに「この橋、通行料いる？」と音声入力で質問。
    2. **現地情報の確認**: AIが即座に「その橋は公的なもので通行は無料のはず」「観光客を狙った非公式な請求の可能性が高い」と回答。
    3. **対処法の提示**: AIが「払う必要はない。『マイ・ミー・タン（お金ないです）』と言って、そのまま通り過ぎてください」と具体的なタイ語のフレーズと行動を指示。
- **結果**: AIの指示通りに行動し、1円も払わずにその場を切り抜けることができた。
- **ログ**: giant_material_database.md (line: 1408-1450)
