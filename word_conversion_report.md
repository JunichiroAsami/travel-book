## 3. Amazonへのアップロード手順

### 3.1. Amazon KDPへのアップロード

**手順**：
1. Amazon KDP（Kindle Direct Publishing）にログイン
2. 「新しい本を作成」をクリック
3. 「電子書籍または印刷本」を選択
4. 「原稿をアップロード」で`complete_manuscript_v10.docx`をアップロード
5. プレビューを確認
6. 出版

### 3.2. PDF化（オプション）

**必要に応じて、Word形式からPDF形式に変換**：

**方法1：Wordで変換**
1. `complete_manuscript_v10.docx`をWordで開く
2. 「ファイル」→「名前を付けて保存」→「PDF」を選択
3. 保存

**方法2：コマンドラインで変換**
```bash
libreoffice --headless --convert-to pdf complete_manuscript_v10.docx
```
